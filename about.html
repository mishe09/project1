About this site

This website is a demostration site for the tutorial series “How to BUILD a Website with HTML” on DigitalOcean’s community platform. Learn how to recreate and customize this site with HTML by following the step-by-step instructions in the tutorial series. Participants will finish an understanding of HTML and website ready to deploy to the cloud. No coding experience necessary!
